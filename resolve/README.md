

# 前言

原版内容可以在其[github库](https://github.com/ruanyf/jstraining)上找到。为便于阅读与实用，我将其内容整理成在线电子书的形式，基础框架也用的是阮老师的教材[《ECMAScript 6 入门》](http://es6.ruanyifeng.com/) 中所用的模板。
